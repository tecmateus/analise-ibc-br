Objetivo do projeto
O objetivo desta análise de dados é fornecer insights valiosos sobre a situação econômica do Brasil, utilizando o Índice de Atividade Econômica do Banco Central (IBC-Br) no período de 2009 a 2011. Através da análise cuidadosa desses dados, buscamos identificar tendências, padrões e anomalias relevantes que nos permitirão compreender melhor o desempenho da economia brasileira nesse período.

Descrição do projeto
Neste projeto, utilizamos a linguagem de programação R e técnicas estatísticas para realizar a análise dos dados do IBC-Br. O código desenvolvido permite realizar diversas operações, como leitura dos dados, visualização gráfica, cálculos estatísticos e identificação de tendências temporais. Utilizamos bibliotecas como ggplot2, dplyr e forecast para auxiliar nas análises.

Esperamos que este projeto contribua para um maior entendimento da situação econômica do país durante os anos de 2009 a 2011, e que os insights obtidos possam auxiliar em futuras tomadas de decisão no âmbito econômico.

Instruções de uso
Para utilizar este código, siga as etapas abaixo:

Faça o download dos dados do IBC-Br no período de 2009 a 2011.
Certifique-se de ter o ambiente R e as bibliotecas necessárias instaladas.
Execute o código no RStudio ou em qualquer outra IDE R.
Siga as instruções no código para carregar os dados, realizar análises específicas e gerar visualizações.
